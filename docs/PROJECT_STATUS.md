## 專案概要

SDPRS（智能防災監測與自動響應系統）採用邊緣運算架構，於颱風期間自動監測玻璃破裂與積水風險：

- **玻璃偵測節點**（Raspberry Pi + 攝影機 + 麥克風）：本地端影像 + 聲音雙重偵測，事件即時上報（警報 JSON <1 秒，MP4 佐證影片隨後補傳）。
- **水泵節點**（ESP32）：水位監測與抽水泵滯後自動控制，MQTT 回報。
- **中央伺服器**（Pi 5 / Docker / Zeabur 雲端三種部署形態）：FastAPI + 繁體中文即時儀表板（WebSocket 推播、聲音警報、監控牆）。
